# Skin - Cosmetics Online Store

This is a cosmetics online store project built with **HTML, CSS and JavaScript**.

I created this project while practicing frontend development. The goal was to build a simple e-commerce interface for cosmetic products, including product cards, prices, a shopping cart section and basic user interaction.

## Project Overview

The website presents a fictional cosmetics store called **Skin**. It includes:

- Product listing section
- Product cards with prices
- “Add to cart” buttons
- Shopping cart area
- Purchase confirmation message
- Footer with store information, help center links and newsletter section

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Practiced

Through this project, I practiced:

- Building a static e-commerce layout
- Structuring product cards
- Styling a visual product catalog
- Creating a cart section
- Using JavaScript for basic interaction
- Organizing a complete frontend project
